# Global EV Sales Analysis | Power BI Dashboard

Power BI Dashboard, Data Analysis & Market Trend Insights

---

## Project Overview

This project focuses on analyzing global electric vehicle (EV) sales data using Power BI to understand regional trends, vehicle category performance, and overall market growth.

The dashboard provides an interactive view of EV adoption across different regions and time periods, helping to identify key patterns and insights in the global EV market.

This project was created as a portfolio work to demonstrate practical skills in data visualization, dashboard design, and analytical thinking using Power BI.

---

## Problem Statement

With the rapid growth of electric vehicles worldwide, understanding regional adoption patterns and category-wise performance is crucial for market analysis and strategic planning.

The objective of this project is to:

- Analyze global EV sales trends over time  
- Compare EV adoption across regions  
- Study category-wise performance (Cars, Buses, Trucks)  
- Present insights through an interactive Power BI dashboard  

---

## Key Analysis Areas

### Regional Analysis
- EV sales comparison across major regions  
- Identification of leading and emerging EV markets  

### Category Analysis
- Sales contribution by vehicle category  
- Regional dominance by vehicle type  

### Time-Based Trends
- Growth patterns across years  
- Latest year performance overview  

---

## Dashboard Highlights

- KPI cards showing total EV sales, EV stock, and latest year  
- Regional EV sales comparison  
- EV growth percentage by region  
- Category-wise EV penetration  
- Developed vs developing regions contribution
